# xs-firestack


# Getting Started

## :arrow_forward: Getting started
Install deps
```bash
curl -sL firebase.tools | bash
brew install java
npm ci
```
Login to firebase
```bash
firebase login
```

## :globe_with_meridians: Environments and Deploy

there is two environment configured in `.firebaserc`
- dev
  - `firebase use dev`
- prod
  - `firebase use prod`
